# Clustering_and_classification_of_households
Кластеризация и классификация домохозяйств с использованием IBM SPSS Statistics.

Задача: определить количество и описать группы, на которые делятся домохозяйства по их финансовой обеспеченности.

Основным источником является набор данных, в котором представлена информация о финансовом состоянии домохозяйств, а именно, о цене, типе, размере жилья, наличии машины, отдельного дома, дополнительной квартиры, доходах и долгах по кредитам и т.д. Источник данных: https://www.hse.ru/rlms/spss. 

Данные представлены в файле data, результаты работы - в файле report.
